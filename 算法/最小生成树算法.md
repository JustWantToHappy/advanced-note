## Prim算法(加点法)
1. 选距离生成树最近节点
2. 最近节点加入生成树
3. 更新非生成树节点到生成树的距离
```python
# 构建prim图，加点法实现
v,e=map(int,input().split())
grid=[[10001 for _ in range(v+1)] for _ in range(v+1)]
# 记录已经联通的节点
union=set()
minDist=[10001 for _ in range(v+1)]
for _ in range(e):
    start,end,weight=map(int,input().split())
    grid[start][end]=weight
    grid[end][start]=weight
 
total=0
 
for _ in range(v-1):
    cur=-1
    minWeight=float("inf")
    for i in range(1,v+1):
        if i not in union and minDist[i]<minWeight:
            minWeight=minDist[i]
            cur=i
    union.add(cur)
    for j in range(1,v+1):
        if grid[cur][j]<minDist[j] and j not in union:
            minDist[j]=grid[cur][j]
     
for i in range(2,v+1):
    total+=minDist[i]
 
print(total)
```
## 克鲁斯卡尔算法(加边法)
1. 将所有边按权值从小到大排序
2. 依次遍历每条边 (u, v, w)：
   - 如果 u 和 v 不在同一连通分量（用并查集判断）
   - 则将该边加入生成树，并合并 u 和 v 的集合
   - 否则跳过（会成环）
3. 直到选了 V-1 条边，算法结束
```python
import heapq
# 构建prim图，加点法实现
v,e=map(int,input().split())
# 记录已经联通的节点
union=set()
roots=[i for i in range(v+1)]
total=0
graph=[]

for _ in range(e):
    start,end,weight=map(int,input().split())
    graph.append((weight,start,end))

graph.sort(key=lambda x:x[0])

def findUnion(v):
    if roots[v] == v:
        return v
    roots[v] = findUnion(roots[v])
    return roots[v]

def isSameUnion(v,w):
    return findUnion(v)==findUnion(w)

# w->v
def join(v,w):
    rootv=findUnion(v)
    rootw=findUnion(w)
    if rootv==rootw:
        return
    roots[rootw]=rootv

for weight,start,end in graph:
    if not isSameUnion(start,end):
        total+=weight
        join(start,end)
# 最小生成树权重之和
print(total)


```
