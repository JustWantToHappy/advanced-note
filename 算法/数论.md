- gcd(最大公约数)
- 辗转相除法：用于求两个整数的最大公约数(古希腊人发明)
```javascript
function gcd(a, b) {
    if (a % b === 0) {
        return b;
    }
    return gcd(b, a % b);
}

```

- 更相减损法：用于求两个整数的最大公约数(中国人发明)

  
```javascript
function gcd(a, b) {
    while (a !== b) {
        if (a > b) {
            a = a - b;
        } else {
            b = b - a;
        }
    }
    return a;
}
```
