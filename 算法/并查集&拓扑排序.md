## 并查集模板
```python
def findUnion(v):
    if parents[v]==v:
        return parents[v]
    '''
    parents[v]=findUnion(parents[v])
    return parents[v]
    '''
    return findUnion(parents[v])
# v->w(v的集合加入到w的集合中)
def join(w,v):
    w=findUnion(w)
    v=findUnion(v)
    parents[v]=w
```

## 拓扑排序模板
1. 找到所有入度为0的节点，统计每个节点入度数量，入度0的节点都加入到queue队列中（实际上是BFS的方式）
2. 拿到队列首节点，遍历首节点相连接的节点，这些节点的入度数量减一,入度为0的节点加入队列。循环以上过程直到队列为空

如何判断DAG是否存在环？
加入到queue的节点数量小于输入节点数即存在环

## 检查DAG(有向无环图)是否存在环
1. DFS访问状态法
2. 拓扑排序

为什么不能使用并查集判断DAG是否存在环？
举个例子：A->B->C这里有两条边：A->B以及B->C，在标准的并查集中，只会认为AB是一个集合，BC是一个集合，而AC不是一个集合。
总结：并查集只适用于无向图环的检测
