## 常见选择器权重表
| 选择器           | 权重      |
| ------------- | ------- |
| `#app`        | (1,0,0) |
| `.item`       | (0,1,0) |
| `[type=text]` | (0,1,0) |
| `:hover`      | (0,1,0) |
| `div`         | (0,0,1) |
| `::before`    | (0,0,1) |
| `:is()`        | 括号里面最高的选择器的权重 |
| `:where()`     | (0,0,0) |
```text
(0,9,1)+(0,9,1)=(0,18,2)<(1,0,0)
```
