## 线性插值公式
假设你有一个简单的数据集，其中两个点 (x1, y1) 和 (x2, y2) 是已知的。现在你想要找到在这两个点之间的某个点 (x, y) 的值，其中 x 是在 x1 和 x2 之间的某个值。

线性插值公式如下：

y = y1 + (y2 - y1) * ((x - x1) / (x2 - x1))
