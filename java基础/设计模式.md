## 适配器模式
```java
interface Interface1{
    void method1();
    void method2();
}

//适配器类没有new的意义，所以使用抽象类
class Interface1Adapter implements Interface1{
    @Override
    public void method1() {}
    @Override
    public void method2() {}
}

//我的类要实现接口中的某一个方法，如果直接实现接口，就需要都重写，这时候我们可以加中间层，也就是适配器类
class Interface1Impl extends Interface1Adapter {
    @Override
    public void method1() {}
}
```
